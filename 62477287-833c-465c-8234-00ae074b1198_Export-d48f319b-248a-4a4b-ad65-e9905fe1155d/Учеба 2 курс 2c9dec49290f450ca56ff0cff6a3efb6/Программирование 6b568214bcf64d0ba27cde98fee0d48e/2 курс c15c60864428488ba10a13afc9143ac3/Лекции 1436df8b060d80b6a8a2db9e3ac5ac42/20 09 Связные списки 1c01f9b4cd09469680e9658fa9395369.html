20.09 Связные списки

Связный список, который на ряду с элементом содержит указатель.

В односвязном списки можно сделать 4-варианта добавления:

  1. В начало
  1. Послде заданного
  1. в определенную позицую
  1. в конец

Доступ (п)
Поиск О(n)
Удаление (аналог remove If) O(n)
Вставка (в начало или в конец) - 0(1)
Удаление (сначала или с конца) (1)
size O(1)
sort O(NlogN)


Минусы
-нельзя быстро обратиться по индексу
Выделение памяти под каждый элемент (накладные расколы)

#include <cstdlib>
#include <iostream>
#include <string>

using namespace std;

struct Node {
  string person;
  Node* next;
};

void addFile(Node* fList, string person) {
  Node* newNode = new Node;
  newNode->person = person;
  newNode->next = fList->next;

  fList->next = newNode;
}

void Delete(Node* node) {
  Node* deleteNode = node->next;
  node->next = deleteNode->next;
}

void printFlist(Node* flist) {
  while (flist != nullptr) {
    cout << flist->person << " ";
    flist = flist->next;
  }
}

int main() {
  Node* forwardList = new Node{"Head", nullptr};
  forwardList->next = new Node{"svinia", nullptr};
  forwardList->next->next = new Node{"xyi", nullptr};
  // cout << forwardList->next->next->person;
  printFlist(forwardList);
}

Двусвязаный список

Стек - линейный список, у которого операция вставки и удаления выполняются только в одном конце. Динам.структура данных. Стек может основан на массиве и на списке(для нас на списке).

Плюсы стека:

  1. Добавление и удаление элемента

Минус:

  1. Ограниченный доступ(по индексу пиздец)
  1. Ограниченная емкость
  1. Не подходит для произвольного доступа к элементу
  1. Переполнение

Очередь плюсы:

  1. ДОбавление элемента и удаление О(1).

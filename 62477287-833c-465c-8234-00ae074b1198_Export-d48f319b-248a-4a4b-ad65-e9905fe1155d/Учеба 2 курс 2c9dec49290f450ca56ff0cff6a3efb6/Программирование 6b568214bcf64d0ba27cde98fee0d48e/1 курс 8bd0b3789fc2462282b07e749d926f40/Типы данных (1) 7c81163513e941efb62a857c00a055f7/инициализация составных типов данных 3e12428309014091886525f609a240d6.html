инициализация составных типов данных

Массивы

Массив - это набор элементов одного типа. Все элементы массива хранятся в смежных блоках памяти, и каждый из них может быть непосредственно доступен по его индексу.

int myArray[5] = {1, 2, 3, 4, 5};

Структуры

Структура - это пользовательский тип данных, который позволяет хранить разные типы данных вместе.

struct Person {
    std::string name;
    int age;
};

Person john = {"John Doe", 30};

Классы

Классы в C++ являются расширением структур и могут содержать функции, знакомые как методы.

class Car {
public:
    std::string model;
    void startEngine() {
        //...
    }
};

Car myCar;
myCar.model = "Toyota";
myCar.startEngine();

Векторы

Векторы в C++ являются динамическими массивами с возможностью изменять размер.

std::vector<int> myVector = {1, 2, 3, 4, 5};
myVector.push_back(6); // добавить элемент в конец вектора

Некоторые методы Вектора

Ряд функций позволяет добавлять элементы на определенную позицию.

  • emplace(pos, value): вставляет элемент value на позицию, на которую указывает итератор pos
  • emplace_back(value): вставляет элемент value конец вектора
  • insert(pos, value): вставляет элемент value на позицию, на которую указывает итератор pos, аналогично функции emplace
  • insert(pos, n, value): вставляет n элементов value начиная с позиции, на которую указывает итератор pos
  • insert(pos, begin, end): вставляет начиная с позиции, на которую указывает итератор pos, элементы из другого контейнера из
    диапазона между итераторами begin и end
  • insert(pos, values): вставляет список значений начиная с позиции, на которую указывает итератор pos

Удаление элементов

  • clear: очищает вектор
  • pop_back(): удаляет последний элемент вектора:
  • erase(p): удаляет элемент, на который указывает итератор p. Возвращает итератор на элемент, следующий после удаленного,
    или на конец контейнера, если удален последний элемент
  • erase(begin, end): удаляет элементы из диапазона, на начало и конец которого указывают итераторы begin и end.
    Возвращает итератор на элемент, следующий после последнего удаленного, или на конец контейнера, если удален последний элемент

Map

Map в C++ это ассоциативный контейнер, который хранит элементы в соответствии с ключом. Ключи уникальны.

std::map<int, std::string> myMap;
myMap[1] = "one";
myMap[2] = "two";

Set

Set в C++ это контейнер, который хранит уникальные элементы в неупорядоченном порядке.

std::set<int> mySet;
mySet.insert(1);
mySet.insert(2);

Tuple

Tuple в C++ это контейнер, который позволяет хранить разные типы данных в одном комплексе.

std::tuple<int, std::string, char> myTuple = {1, "one", 'a'};

Pair

Pair в C++ это контейнер, который позволяет хранить два элемента разных типов данных как одну единицу.

std::pair<int, std::string> myPair = {1, "one"};

Ассоциативный массив - это тип данных, который позволяет хранить пары "ключ-значение". В отличие от обычного массива, где доступ к элементам осуществляется по их индексу, в ассоциативном массиве к элементам можно обращаться по их ключу. Это особенно полезно, когда нужно быстро найти значение по определенному ключу. В C++ ассоциативные массивы представлены контейнерами типов map и unordered_map.

Кроме ассоциативных массивов, в C++ также есть последовательные контейнеры (например, vector, list, deque, array), адаптеры контейнеров (например, stack, queue, priority_queue) и ассоциативные контейнеры с уникальными ключами (set и unordered_set).
